Decentralized Index Funds Project

Cross-chain, crowd-managed and decentralized index funds


What is Decind Project?

The idea behind the Decentralized Index Funds Project is to use decentralized finance exchanges with smart contracts to replicate indexes on most valued cryptocurrencies.

Also, every people in the world have different reasons to save money: buy a house, prepare the futures of their children, have enough money to retire peacefully, or just make it grows. For each project, their is a different goal time and a different risk to be taken, that should be leveraged by index funds based on their risks and profits. Each project should represented by a different portfolio.

  • This project needs to be trustless and decentralized
  • This project needs to be as much personalized as possible
  • This project needs to have the lowest fees

The final goal is to give access to an investment product in the crypto world that is cheap and simple, for everybody. For this goal to become a reality, not only this needs well-designed products, but also a need for simple investment tools for non-tech people.
